# Notebooks

This directory contains Jupyter notebooks for experimentation and development.

## Notebooks

- `00_getting_started.ipynb` - Introduction and basic setup
- Add more notebooks as you experiment!

## Tips

- Use clear naming conventions (e.g., `01_feature_name.ipynb`)
- Add markdown cells to document your thought process
- Keep notebooks focused on specific topics or experiments
- Export important findings to the `src/` directory as modules
